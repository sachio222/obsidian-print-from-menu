# Print from Menu

Adds a **Print...** option to the right-click context menu for markdown files. Renders the note as clean, styled HTML and opens the system print dialog — no raw markdown, no export steps.

## Usage

1. Right-click any markdown file (in the file explorer or on a tab)
2. Click **Print...**
3. The system print dialog opens with a rendered version of your note

You can print to paper or use "Save as PDF" from the print dialog.

## Features

- Renders headings, tables, code blocks, blockquotes, lists, and images
- Clean print-friendly stylesheet (no dark mode artifacts)
- Title automatically included at the top
- Also available via command palette: **Print current note**

## Installation

Copy the plugin folder to `.obsidian/plugins/print-note/` and enable it in Settings > Community Plugins.
